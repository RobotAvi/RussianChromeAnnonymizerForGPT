## Установка

1. Скачайте нужную версию zip архива
2. Распакуйте архив
3. Откройте Chrome и перейдите в `chrome://extensions/`
4. Включите "Режим разработчика"
5. Нажмите "Загрузить распакованное расширение"
6. Выберите папку с распакованными файлами
